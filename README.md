# springboot-admin
springboot-admin 多模块项目, 父模块监控到子模块下线，给父模块设置的邮箱发送邮件

### 功能介绍

（1）父模块监控到子模块下线，给父模块设置的邮箱发送邮件
（2）父模块：http://127.0.0.1:8080
